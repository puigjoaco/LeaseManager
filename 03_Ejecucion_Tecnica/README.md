# Ejecucion tecnica - LeaseManager

Esta carpeta convierte el set documental final en una capa de ejecucion para construir el sistema sin alterar el set vigente.

Fuente de verdad funcional:

- [PRD_CANONICO.md](../PRD_CANONICO.md)
- [MATRIZ_GATES_EXTERNOS.md](../MATRIZ_GATES_EXTERNOS.md)
- `ADR_ARQUITECTURA_001`, `002`, `003`, `004`, `005`, `006`, `008`

Objetivo de esta carpeta:

- ordenar modulos y dependencias;
- definir fases tecnicas;
- bajar el nucleo a backlog ejecutable;
- fijar criterios de salida por milestone.

Orden recomendado de uso:

1. leer [ROADMAP_TECNICO.md](./ROADMAP_TECNICO.md)
2. revisar [MODULOS_Y_DEPENDENCIAS.md](./MODULOS_Y_DEPENDENCIAS.md)
3. ejecutar [BACKLOG_FASE_1.md](./BACKLOG_FASE_1.md)
4. preparar [BACKLOG_FASE_2.md](./BACKLOG_FASE_2.md) conforme cierre Fase 1
5. usar [CRITERIOS_DE_SALIDA.md](./CRITERIOS_DE_SALIDA.md) para decidir paso entre hitos

Regla de trabajo:

- esta carpeta no redefine producto;
- si aparece una contradiccion con el set activo, manda el set activo y debe reemitirse la ejecucion tecnica.
